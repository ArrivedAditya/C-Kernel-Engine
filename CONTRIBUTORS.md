# C-Kernel-Engine Contributors

CKE recognizes people for reproducible engineering contributions, including
compatibility findings, tests, documentation, kernels, compiler work, and
research evidence. The Git commit and pull-request history remains the
authoritative record; this file is a curated index, not an ownership registry.
