# C-Kernel-Engine Licensing Status

## Current Status

C-Kernel-Engine does not currently publish a root `LICENSE` file. Repository
visibility is not the same as an open-source licence, and licences attached to
third-party submodules or imported projects do not license CKE's original
code.

Until an explicit root licence is adopted, do not assume a general grant to
copy, modify, redistribute, sublicense, or sell CKE code beyond rights supplied
by applicable law or the hosting platform.

## Contributions And Copyright

Git commits, pull requests, `CONTRIBUTORS.md`, `.mailmap`, and release notes
preserve engineering attribution. Attribution is not the same as copyright
ownership or a software licence.

CKE has not adopted a Contributor License Agreement (CLA) or Developer
Certificate of Origin (DCO). A merged pull request therefore does not, by
itself, document a copyright assignment to the maintainer or consent to every
possible future project licence. A contributor must have the right to submit
their material, including code drafted with AI assistance or adapted from a
third party.

## Third-Party Components

Submodules, vendored projects, model files, datasets, generated artifacts, and
external libraries retain their own licences and notices. Their inclusion in
the repository does not change those terms and does not establish the licence
for CKE's original source code.

## Before A Stable Licensed Release

The project should complete these steps before presenting CKE as an
open-source or source-available licensed release:

1. Audit the repository boundary and third-party notices.
2. Select and add an explicit root software licence.
3. Define inbound contribution terms, such as a DCO or a narrowly scoped CLA.
4. Obtain any contributor permissions required for the selected licence.
5. Document how future proprietary components, including possible distributed
   orchestration, remain separated from the licensed CKE core.

This document records project policy and current repository state. It is not
legal advice and is not itself a software licence.
